City Pulse

City Pulse is a project I started with a team of guys from Chicago. The site was designed to give the user a digital visualization of the city of Chicago. We used six APIs that supplied us with geolocated data that we could plot using google maps. This project incorporates web sockets for persisting data from the server to the client in a free flow without a trigger. the site also incorporates the use of cron tasks that fetch fresh data from APIs to be held and then displayed to our users. we chose to retrieve our data via cron rather than on a user action to reduce the rate of API calls from the current amount of users to a constant 15 a minute. Unfortunately this project is no longer hosted.(Heroku is getting a bit expensive.)

Ruby- This code snippet is from a project where I was implementing an html parser in ruby. I had a situation where I had to split an html element at a certain word, but given the fact that the html elements are often nested I had to develop a recursive method that would divide each html element that this word was encompassed by until it reached an element who's content contained only text.

  #this is a method on a class that is called paragraph
  #paragraph class iherits from the element class
  #the idea is that if a paragraph is too large we call this method and split it at the paragraphs splitting point
  #this recursively calls split_element in the event that the splitting point falls upon another element in the current element
  
  #called on the paragraph class that inherits from the element class 
  
  def split_paragraph
    element_children.each do |element|
      if words_left_in_current >= element.word_count
        current.add_child(element.nokogiri_object)
      elsif element.name == 'text'
        split_text_node(element)
      else
        element.split_element(words_left_in_current).each_with_index do |half, index|
          pair[index].add_child(half.nokogiri_object)
        end
      end
    end
    pair
  end
 
  #called on the element class 

  def split_element(words_left)
    element_children.each do |element|
      if words_left >= element.word_count
        current(words_left).add_child(element.nokogiri_object)
        words_left -= element.word_count
      elsif first_card_is_full == true
        pair[1].add_child(element.nokogiri_object)
      elsif element.name == 'text'
        split_text_node(element, words_left)
        self.first_card_is_full = true
      else
        split_and_add_child(element, words_left)
      end
    end
    pair
  end

Rails- This is a rails model validation that is checking html to see if it violates a sanity check for displaying in our UI. If the html violates the criteria check we save the violation in the database and flag the html as needing attention.


  def too_many_cards?
    card_array.count > card_limit
  end

  def card_limit
    256
  end

  def div_too_long?
    card_array.select { |section| section.content.length > 1351 }.count > 0
  end

  def no_images?
    nokogiri_content.css('img').empty?
  end

  def validate_cardified_content
    if cardified_content
      fail ArticleErrorTooManyCards.new(cardified_content, self) if too_many_cards?
      fail ArticleErrorDivTooLarge.new(cardified_content, self) if div_too_long?
      fail ArticleErrorNoImages.new(cardified_content, self) if no_images?
    end
  end

  def check_cardfied_content
    begin
      validate_cardified_content
    rescue => e
      self.needs_attention = true
      self.needs_attention_reason = e.reason
      self.cardified_content = cardified_content_was
    end
  end
