Projects

City Pulse

City Pulse is a project I started with a team of guys at Dev Bootcamp as our capstone. The site was designed to give the user a digital visualization of the city of Chicago. We used six APIs that supplied us with geolocated data that we could plot using google maps. This project incorporates web sockets for persisting data from the server to the client in a free flow without a trigger. the site also incorporates the use of cron tasks that fetch fresh data from apis to be held and then displayed to our users. we chose to retrieve our data via cron rather than on a user action to reduce the rate of API calls from the current amount of users to a constant 15 a minute.

DBC Power Rankings

DBC Power Ranking is another project I a was able to be a part of at DBC. This was our first rails application and gave us a chance to learn rail's server actions and routing. This project was developed to gamify the mentor process here at DBC. The app uses an oauth system and an API set up by former students that verifies a user as a DBC student and correlates them with their data pulled from the DBC API. The app gives students the ability to give points to fellow students for their help and resources. This was the first project we were able to work with Rspec and develop controller tests and integration tests.
